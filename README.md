# 💪 BMI Calculator (Python)

A simple GUI-based BMI calculator using Python and Tkinter.

## 🚀 Features

* Calculate BMI instantly
* Health category detection
* Clean dark UI

## ▶️ Run

```
python bmi.py
```

## 📊 Categories

* Underweight
* Normal
* Overweight
* Obese
